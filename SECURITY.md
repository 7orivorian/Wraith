# Security Policy

## Supported Versions

| Version   | Supported          |
|-----------|--------------------|
| \>= 3.0.0 | :white_check_mark: |
| 2.0.1     | :white_check_mark: |
| <= 2.0.0  | :x:                |

## Reporting a Vulnerability

Vulnerabilities can be reported by visiting the 'Security'
tab on this repo and clicking 'Report a vulnerability'.
